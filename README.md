# Text2Speech

This is a Speech to Text Converter (Text to Speech's continuation - Reverse project) made with HTML, CSS, JavaScript and Tailwind CSS that simply runs on Web Speech API.

## Technologies Used

- HTML
- CSS
- JavaScript
- Tailwind CSS
- Web Speech API
